## Run Locally

Clone the project

```bash
  git clone https://github.com/Dhirajmathankar/Ecommerce/
```

Go to the project directory

```bash
  cd Ecommerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
